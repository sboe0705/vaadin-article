The User Library "JSF 2.2 (Mojarra 2.2.0)" has to be imported to run the application properly.

- Open Eclipse preferences (Windows -> Preferences)
- Navigate to User Libraries (Java -> Build Path -> User Libraries)
- Press "Import..." and select "JSF 2.2 (Mojarra 2.2.0)" (should be found automatically, otherwise select manually from the directory "libraries")
